# **App Name**: TN.proto

## Core Features:

- Employee Clock-In/Out: Allows employees to record their work hours with clock-in/out functionality.
- Daily Work Logs: Enables employees to create and submit daily logs detailing their tasks and accomplishments.
- Admin Employee Management: Provides an interface for admins to create, manage, and delete employee accounts with assigned employee codes and hashed passcodes.
- Attendance Tracking: Enables admin users to view a dashboard of when employees checked in.
- Admin Notice System: Allows admins to post public announcements or private notices to individual employees, managed through an admin panel.
- Error Logging: Automatically captures and logs API and UI runtime errors, viewable on a dedicated admin page.
- Credential Verification: AI tool evaluates inputted information for logging in to verify users successfully entered a valid employee code and passcode by comparing inputted information to a database.
